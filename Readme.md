🚀 CP-U: A Website to Make Your Mind Run Like a CPU

Solve. Learn. Grow. Compete.

CP-U is a full-stack platform designed to be the ultimate all-in-one resource for the competitive programming community. It's built for coders who want to track their progress, learn new algorithms, solve challenging problems, and grow together.

Inspired by the giants:

Codeforces & AtCoder (for the competitive spirit)

CSES & cp-algorithms.com (for the deep-dive learning)

"Competitive Programming 4" (for the structured curriculum)

📋 Project Proposal (Evaluation - Phase 1)

CP-U is the foundation for an all-in-one competitive programming platform. It provides a central hub for coders to track progress, compete, learn, and collaborate.

Core Features (Phase 1)

Full-Stack Architecture: Built with a React frontend (deployed on Vercel), a Node.js/Express backend (on Render), and a PostgreSQL database (on Render).

User Authentication: Secure signup and login system using bcrypt for password hashing and JWT (JSON Web Tokens) for session management.

Community Blog: A complete CRUD feature allowing authenticated users to write, publish, and read technical blogs and articles.

Algorithm Library: A static, growing library of algorithm explanations and reading materials.

🛠 Tech Stack

Frontend: React (Vite), Tailwind CSS

Backend: Node.js, Express

Database: PostgreSQL

ORM: Prisma

Deployment: Vercel (Frontend) & Render (Backend + Database)

✨ Current Features (Working)

Secure User Authentication: Users can create an account, log in, and receive a secure JWT token for a persistent session. Passwords are fully hashed using bcrypt.

Community Blog: Authenticated users can create, read, update, and delete their own blog posts.

Static Algorithm Pages: A "Learn" section with foundational guides on topics like Big O Notation and basic data structures.
